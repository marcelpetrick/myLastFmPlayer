myLastFmPlayer Documentation
============================

``myLastFmPlayer`` is a Linux desktop application for fetching a user's
public Last.fm loved tracks, resolving them through YouTube search, downloading
audio files, and playing local tracks through PyQt.

Start with :doc:`architecture` for the runtime flow and diagrams. Use
:doc:`api` for class and function reference pages generated from the Python
source.
